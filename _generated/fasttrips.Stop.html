get_trips_arriving_within_time(assignment_date, latest_arrival, time_window)¶

Return list of [(trip_id, sequence, arrival_time)] where the arrival time is before latest_arrival but within time_window.

Parameters:
  • assignment_date (a datetime.date instance) – Date to use for assignment (datetime.timedelta requires datetime.datettime instances, and can’t just use datetime.time instances.)
  • latest_arrival (a datetime.time instance) – The latest time the transit vehicle can arrive.
  • time_window (a datetime.timedelta instance) – The time window extending before latest_arrival within which an arrival is valid.

routes = None¶

These are the fasttrips.Route instances that this stop is a part of

stop_id = None¶

unique stop identifier

tazs = None¶

These are TAZs that are accessible from this stop. This is a dict mapping a taz_id to (walk_dist, walk_time) which are in miles and minutes respectively.

transfers = None¶

These are stops that are transferrable from this stop. This is a dict mapping a destination stop_id to (transfer_distance, transfer_time), where transfer_distance is in miles and transfer_time is a datetime.timedelta instance. TODO: This can be a dict, just making it an ordered dict for testing

trips = None¶

These are the trips this stop is a part of This is a list of (trip_id, sequence, arrival time, departure time) Use Stop.TRIPS_IDX_TRIP_ID, Stop.TRIPS_IDX_SEQUENCE, Stop.TRIPS_IDX_ARRIVAL_TIME and Stop.TRIPS_IDX_DEPARTURE_TIME for access.
