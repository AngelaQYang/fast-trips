fasttrips.TAZ¶

class fasttrips.TAZ(input_dir)[source]¶

TAZ class.

One instance represents all of the Transportation Analysis Zones as well as their access links.

Stores TAZ information in TAZ.zones_df, an instance of pandas.DataFrame and access link information in TAZ.access_links_df, another instance of pandas.DataFrame.
